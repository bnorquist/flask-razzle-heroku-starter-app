# Starter App for easy deployment on Heroku
* Razzle Frontend
* Flask backend w/ postgres through SQLAlchemy

## local development instructions
* docker-compose up
* point browser to `http://lvh.me/`
* adding requirements

## Deployment
### Client
* `make deploy.web`

### Server
* `make deploy.api`
